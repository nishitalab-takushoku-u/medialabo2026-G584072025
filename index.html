数当てゲーム

（このテーマの第3回から数当てゲームを作りますので、そのあとにゲームの概要を説明してください。）（200文字以上）

数当てゲームの制作では、JavaScriptを使って、実際に動くプログラムを作る練習をしました。このゲームは、コンピュータがランダムに決めた数字をプレイヤーが当てるものです。数字を入力したときに「もっと大きいですよ」とか「もっと小さいですよ」というヒントを画面にすぐ表示する仕組みを作りました。これを体験したことで、プログラミングの基本である「変数」や、条件によって動きを変える「if文」の使い方をしっかりと理解することができました。また、ただ動くだけでなく、遊ぶ人が使いやすくなるような工夫もしました。例えば、ゲームを最初からやり直すボタンを作ったり、何回目で当たったかをカウントして表示する機能を追加したりしました。コードが思った通りに動いて、ゲームとしてちゃんと遊べたときは本当に嬉しかったですし、プログラミングの楽しさを実感できました。

数当てゲームのページ：kazuate.html

各授業回の学習事項

第1回 WWW と HTML

(今回学習した事柄を説明してください)(100文字以上)

第1回の授業では、HTMLの基本構造について学習しました。ブラウザがどのようにWebページを表示するのかを理解し、h1〜h6タグ、pタグ、imgタグなどの基本的な要素を使って文書を作成する方法を習得しました。自分で書いたコードがブラウザに表示されたときは、ウェブ制作の第一歩を踏み出せた気がして楽しかったです。

演習の成果： 演習1-4:HTMLのさまざまな要素

第2回 CSS による Webページの装飾

第2回の授業では、CSSを使ってウェブページをデザインする方法を学びました。文字の色や大きさを変えたり、背景に色をつけたりする方法を習得しました。HTMLだけで書いたシンプルな文章が、CSSを使うことで一気にきれいで見やすくなることが分かり、デザインの楽しさを実感しました。

演習の成果： ウェブブラウザの紹介

第3回 JavaScript の基本

第3回の授業では、プログラミングのJavaScriptについて学びました。データを保存する「変数」や、条件によって動きを変える「if文」などの基本を習得しました。今までのHTMLやCSSとは違って、頭を使って計算や仕組みを考えるのが難しかったですが、動くページを作るための大切な基礎を学べました。

演習の成果： JavaScript のオブジェクト

第4回 DOM と JavaScript

v第4回の授業では、JavaScriptを使ってHTMLの文字や色を自由に書き換える方法（DOM操作）を学びました。ボタンをクリックしたときに画面が変化するような、使う人とやり取りができるページの作り方が分かりました。これでウェブサイトに色々な仕掛けを作れるようになりました。

演習の成果： 足し算ページ
