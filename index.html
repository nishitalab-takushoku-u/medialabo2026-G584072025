数当てゲーム

（このテーマの第3回から数当てゲームを作りますので、そのあとにゲームの概要を説明してください。）（200文字以上）

本プログラムは、JavaScriptを用いたイベント駆動型プログラミングの実装例である。コンピュータが Math.random によって生成した1から10までの乱数を正解（kotae）とし、プレイヤーがテキストボックスから入力した数値（yoso）を比較・判定する仕組みとなっている。 技術的な特徴として、まず addEventListener を用いて「回答」ボタンにクリックイベントを登録し、関数 hantei を実行する。関数内では、document.querySelector によりDOM（Document Object Model）を操作し、入力値の取得およびページ上の表示（試行回数やヒント）を動的に更新している。 特に条件分岐（if...else）の設計においては、試行回数 kaisu を優先的に判定するロジックを採用した。プレイヤーには最大3回のチャンスが与えられ、各試行ごとに「正解」「大きい」「小さい」のフィードバックを即座に提供する。3回失敗した際には正解を明示し、4回目以降の入力に対しては「すでにゲームは終了している」というメッセージを表示して入力を制限する仕様となっており、堅牢なユーザーインターフェースを実現している。

数当てゲームのページ：kazuate.html

各授業回の学習事項

第1回 WWW と HTML

(今回学習した事柄を説明してください)(100文字以上)

第1回の授業では、HTMLの基本構造について学習しました。ブラウザがどのようにWebページを表示するのかを理解し、h1〜h6タグ、pタグ、imgタグなどの基本的な要素を使って文書を作成する方法を習得しました。Web制作の第一歩として、正しいタグの使い方を学ぶことができました。

演習の成果： 演習1-4:HTMLのさまざまな要素

第2回 CSS による Webページの装飾

第2回の授業では、CSSを用いたWebページのデザイン方法を学びました。文字の色やサイズ、背景色の変更といった基本的な装飾から、ボックスモデルの概念、セレクタの使い方まで習得しました。HTMLで作成した文書構造を維持しつつ、見た目を整えることの重要性を実感しました。

演習の成果： ウェブブラウザの紹介

第3回 JavaScript の基本

演習の成果： JavaScript のオブジェクト

第3回の授業では、プログラミング言語であるJavaScriptの基礎を学びました。変数、データ型、演算子、そして条件分岐（if文）やループ処理（for文）といった制御構造について理解を深めました。静的なHTMLに動きを与えるための論理的な考え方を学び、簡単な計算処理やコンソールへの出力ができるようになりました。

第4回 DOM と JavaScript

本日はJavaScriptを用いたイベント駆動型プログラミングの基礎と、DOM操作による動的なWebサイトの構築方法を学習した。具体的には、input要素からユーザーの入力値を取得し、addEventListenerを使用してボタンクリック時に判定処理を行う仕組みを習得した。 また、「数当てゲーム」の実装を通じて、条件分岐（if...else）による論理構造の設計を実践した。特に、試行回数を制限するカウント処理や、textContentを用いたリアルタイムな情報更新を行うことで、ユーザーインターフェースの双方向性を高める手法を深く理解することができた。この学習により、HTMLの静的構造にJavaScriptで動的な機能を付与する一連の流れをマスターした。

演習の成果： 足し算ページ
